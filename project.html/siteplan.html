Site Plan: Local Services Hub

Site Name

Local Services Hub

This name reflects a central place to discover, share, and review local services in the community.

Site Purpose

The website helps users discover reliable local services (plumbers, electricians, tutors, etc.), read reviews, and submit their own feedback to build community trust.

Scenarios

  • 1. A user wants to find a highly-rated plumber nearby and read past reviews.
  • 2. A user wants to leave feedback for an amazing gardener they hired recently.

Color Scheme

Color palette selected using Coolors.

  • Primary: #2f4858 (Dark Blue-Grey)
  • Accent: #f4a261 (Warm Orange)
  • Background: #f9f9f9 (Light Gray)
  • Text: #333333 (Dark Gray)

Typography

  • Heading Font: Segoe UI, Tahoma, sans-serif
  • Body Font: Segoe UI, Tahoma, sans-serif
